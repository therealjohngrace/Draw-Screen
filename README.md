# Draw-Screen
A tool for drawing on your windows home screen, games, and more.
This tool allows you to draw with a click. Need to memorize something on a game without Notepad? Draw-Screen got your back. You can even use it to memorize pictures!
# How to Use
In the installation process of Draw-Screen, the Setup Wizard will ask you if you would like to create a shortcut for Draw-Screen. It is reccomended you press Yes to make your experience better. Once that's done, you can click the tool, adjust the size of the brush, switch brushes, and change colors.
## Saves
Saves are for saving specific drawings. Like notes for a specific change in a game so you don't mess up. Saves are a beautiful way of saving your hands.
